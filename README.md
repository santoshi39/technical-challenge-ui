# technical-challenge-ui

Frontend App in Vuejs 

-User registration Page

-Login Page

-Logout Page

-Upload file Page

-View data Page

-Home Page

-Dashboard Page

## Installation ⚒️

1. Install all packages

   ```
   # npm install [for npm]
   ```

2. Run development server

   ```
   # npm run serve [for npm]
   ```

4. Generate build files for deployment

   ```
   # npm run build [for npm]
   ```
